Welcome to our El Paso Bus webpage! Here's a quick guide to the four buttons on the right top:


  • Current Routes -> Click to view the current bus routes system in El Paso with their current ridership.
  • Predict Model -> Click to view our prediction ridership model in hexbin format. Click on each hexbin to see the ridership data and some social data.
  • Import Routes -> Click to import routes from your local device. The ridership will be calculated based on which hex bins the routes crossed.
  • GitHub Pages -> Click to visit our GitHub page if you find our project interesting.

Thank you for using our webpage!
